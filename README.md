
  # DX 추진사업단 홈페이지 

  DX 추진사업단 홈페이지 ( https://opensource.khu.ac.kr ) 

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  Run `npm run build` to build the production version, then copy the files in the `build` directory to the server.

  Run `npm run start` to start the production server.
